# Программирование микроконтроллеров и датчиков СКУД

Здесь собраны все необходимые материалы, схемы, код и наглядные примеры для выполнения лабораторных и практических заданий.

## 🚀 Как начать работу?

Весь курс построен на использовании симулятора **Wokwi**.
1.  Убедитесь, что у вас есть аккаунт на [Wokwi.com](https://wokwi.com)
2.  Рекомендуемый старт: Практическое задание 1 и 2

## 📁 Структура репозитория

Все задания разбиты по тематическим папкам. Каждая папка — это отдельная практическая работа или проект.

*   **`Практическая работа 01/`** — Основы работы с GPIO.
*   **`Практическая работа 02/`** — Работа с кнопками и прерываниями.
*   **`Практическая работа 03/`** — Подключение датчиков (например, HC-SR501, RC522).

### Что внутри каждой папки?

В папке с заданием есть:
*   **`name_word.doc`** - практическое задание.
*   **`name_gif.gif`** — **Важно!** Анимированная GIF-ка, демонстрирующая, как именно должна работать готовая программа.
